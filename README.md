# blog-in-clojure

Simple blog, written in Clojure and Noir. 
Just a "Hello, world!" in Clojure.

## Environment

* Java 1.6 +
* Clojure
* MongoDB

## Usage

If you use cake, substitute 'lein' with 'cake' below. Everything should work fine.

```bash
lein deps
lein run
```
